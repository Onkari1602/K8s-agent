Intelligent Autoscaling

Scale UP When CPU utilization > 85% AND trend is increasing for 15+ minutes. Adds 1 replica (max 5).
Scale DOWN When CPU & Memory < 20% for 3 consecutive cycles (15 min). Removes 1 replica (min 1).
Trend Detection Compares last 15 min vs previous 15 min. Detects increasing, decreasing, or stable patterns.
Predictive Scales BEFORE hitting limits by detecting usage acceleration trends.

Auto Right-Sizing

Detection Tracks P95 CPU/Memory usage per deployment over 30 min window.
Trigger If P95 usage < 20% of requests for 3 consecutive cycles (15 min).
Action Sets new requests to P95 x 2.5 (CPU) and P95 x 1.5 (Memory). Min: 10m CPU, 64Mi Memory.

Node Consolidation

Detection Calculates free CPU/Memory across all nodes in each node group.
Trigger If total free capacity > 1 full node's capacity.
Action Scales down node group by removing the extra node(s). Kubernetes drains pods automatically.
